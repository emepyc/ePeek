What is e!Peek?

1. A Javascript plug-in

e!Peek as a Javascript plug-in

  • A minimal genome browser
  • Written in Javascript as a plug-in
  • Uses D3.js for rendering
  • Based on the Ensembl REST API
  • Simple and easy to insert in any web page:
    • Gene pages?
    • Blog posts?
    • Wikipedia articles?
  • A learning project written for fun
    • The Ensembl REST API
    • Javascript / D3 / HTML / SCSS...
    • Web technology

e!Peek as a Javascript plug-in

<div id="genomeBrowser"></div>

<!-- Remote D3.js -->
<script src="http://d3js.org/d3.v3.min.js"></script>

<!-- Remote e!Peek JS -->
<script src="http://www.ebi.ac.uk/~mp/ePeek.min.js"></script>

<!-- Remote e!Peek CSS -->
<link rel="stylesheet" href="http://www.ebi.ac.uk/~mp/ePeek.css" type="text/css" />

<script>
  (funcion() {
     var gB = genomeBrowser().species("human").gene("BRCA2");
     gB(document.getElementById("genomeBrowser"));
  })();
</script>
