e!Peek - The Minimal Genome Browser

What is e!Peek?

e!Peek is a javascript plug-in that allows you to insert a simple genome browser in any web page or document where javascript is allowed.

It is very simple to embed and configure. You only need to insert a few javascript lines in your page and that's all. Check the e!Peek documentation for more details.

e!Peek is written in javascript, using the D3 library for svg rendering and is powered by the Ensembl REST API

How does it look like?

Well, check for yourself! The genome track is zoomable and pannable. You can jump to orthologues, etc... And you can link to the e!Peek iOS compatible web app through the "QR code" link

The plug-in is easily customizable and options can be added or removed easily based on the host requirements. For example, the search box, information about homologues (orthologues/paralogues), etc. can be presented or hided.
