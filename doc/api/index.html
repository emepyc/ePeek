API Reference

e!Peek

Genome Browser
  • genome.start: Starts the plugin using the defined coordinates
  • genome.right: Pans the genome browser to the right
  • genome.left: Pans the genome browser to the left
  • genome.zoom: Zooms in/out the genome browser
  • genome.allow_drag: Sets the response of the genome browser to mouse panning / zooming
  • genome.height: Gets/Sets the height of the genome browser
  • genome.width: Gets/Sets the width of the genome browser
  • genome.background_color: Gets/Sets the background color of the genome browser
  • genome.foreground_color: Gets/Sets the foreground color of the genome browser
  • genome.ease: Gets/Sets the d3.ease function to use in the genome browser
  • genome.homologues: Retrieves homology information from the REST API
  • genome.pin: Places marks in the genome browser
  • genome.species: Gets/Sets the species to be used in the genome browser
  • genome.chr: Gets/Sets the chromosome (or seq_region) to be used in the genome browser
  • genome.from: Gets/Sets the start coordinate to be used in the genome browser
  • genome.to: Gets/Sets the end coordinate to be used in the genome browser
  • genome.gene: Gets/Sets the gene name to be used in the genome browser
  • genome.genes: Gets the genes currently displayed in the genome browser
  • genome.gene_info_callback: Callback called every time a gene is selected
  • genome.gene_search_callback: Callback called every time a gene is searched via the REST API
  • genome.ensgene_search_callback: Callback called every time an Ensembl gene ID is searched
  • genome.genes_callback: Callback executed after every pan / zoom event has finished
  • genome.tooltip: Creates tooltips based on epeek.tooltip

Tooltips
  • epeek.tooltip: Gets a new tooltip
  • epeek.type: Gets/Sets the type of information the tooltip will display
  • tooltip.background_color: Gets/Sets the background color for the tooltip
  • tooltip.foreground_color: Gets/Sets the foreground color for the tooltip

Ensembl REST API
  • rest.call: Makes an asynchronous call to the ensembl REST API
  • rest.url.region: Retrieves the genes included in the specified region
  • rest.url.species_gene: Retrieves the ensembl gene associated with the given name and species
  • rest.url.homologues: Retrieves the homologues (orthologues + paralogues) of the given ensembl ID
  • rest.url.gene: Retrieves the ensembl gene associated with the given ensembl ID
  • rest.url.chr_info: Retrieves the information associated with the given seq_region
  • rest.limits: Gets the limits imposed by the Ensembl REST API

Tree
  • epeek.tree: Creates a new tree object
  • tree.tree: Gets the created tree
  • tree.find_node_by_name: Finds and return a node of the tree by name
  • tree.newick: Returns the newick string of the tree
  • tree.lca: Returns the lowest common ancestor of a list of nodes
  • tree.apply: Given a node of the tree and a callback, apply the callback to all subnodes
  • tree.is_leaf: Returns true on leaf nodes or false otherwise
  • tree.depth: Returns the depth of a given node in the tree (starting from the root)
  • tree.name: Returns the name of a given node
  • tree.children: Returns the children of a given node
  • tree.parent: Returns the parent of a given node
  • epeek.species_tree: Converts a newick string into a tree
  • species_tree.update: Updates the species_tree with new data
  • species_tree.background_color: Sets the color for inactive nodes
  • species_tree.foreground_color: Sets the color for active nodes

Misc Methods
  • epeek.scriptPath: Retrieves the path to the file named as its argument
  • selection.moveToFront: Moves front d3.selections
  • selection.watch: Sets a "watcher" to a given style property of a d3.selections
  • selection.selectAncestor: Retrieves the first ancestor of the selected node of a given type
