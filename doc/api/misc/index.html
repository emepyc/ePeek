Misc Methods

# epeek.scriptPath(<string>)

This method returns the full path of a given script file. The script file has to be imported in the current page. It can be used to load resources relative to a given script file.

Example:


var path = epeek.scriptPath("name_of_file.js");
// ...

some_element
   .append("img")
   .attr("src", path + "relative/path/to/image.png");

// ...

#D3 methods

The epeek plug-in relies on the d3 for all the visual elements created. Here we describe several utility functions defined by the epeek library to work with D3 selections

# selection.moveToFront ()

This method moves to the front all the nodes in the selection

# selection.watch (<string, callback>)

This method sets a "watch" to a given style property on a selection. This means that, for a given selection, the callback given as the second argument is fired if the style property defined as the first argument changes. The callback is called on the selection passing the old and new values of the property. For example:


d3.select(div)
  .watch("width", function(oldVal, newVal) {
     // Here, "this" is the selected element
  });

# selection.selectAncestor (<string>)

Given a node retrieves its first ancestor having the type specified by the argument. For example:


var div = d3.select("#node").selectAncestor("div");
