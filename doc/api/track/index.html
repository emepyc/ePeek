Board

tnt.board creates a new track-based visualization. Typically, each track displays a set of features defined during their setup. Boards can be panned or zoomed within their defined limits. New boards are created using the tnt.board method. Once it is created new tracks can be added with tnt.board.add_track

{% highlight javascript linenos %} var myBoard = tnt.board().from(0).to(500).left(0).right(500); var locationTrack = tnt.board.track() .height(20) .background_color("white") .display(tnt.board.track.feature.axis()) .orientation("top") ); var blockTrack = tnt.board.track() .height(30) .background_color("#FFCFDD") .data (tnt.board.track.data() .update( tnt.board.track.data.retriever.sync() .retriever(function () { return [ { start : 200, end : 350 } ] } {% endhighlight %}

Each new track needs to know how to get the data to display (this can by synchonously or asynchronously). A new call to get new data is performed everytime the board changes its state (while panning / zooming or programmatically). This allows the client application to only deal with the data that is meant to be shown on screen (imagine a genome browser that has to keep in memory the whole set of genomic features to display in, say, a whole chromosome!). Each track should also define how to represent each of the features to display. TnT offers predefined common displays (like rectangles, lines, areas, etc) ready to use. New displays can also be created using the d3 library. Composite displays are also allowed and a very powerful tool for displaying complex data. Many examples of how to create different visualizations can be seen in the examples (BROKEN LINK) page. A minimal example would look like this:

{% highlight javascript linenos %} var myBoard = tnt.board().from(20).to(500) myBoard(document.getElementById("mydiv")); myBoard.right (1000); var location_track = tnt.board.track() .height(20) .background_color("white") .display(tnt.board.track.feature.axis() .orientation("top") ); var block_track = tnt.board.track() .height(30) .background_color("#FFCFDD") .data(tnt.board.track.data() .update( tnt.board.track.data.retriever.sync() .retriever (function () { return [ { start : 200, end : 350 } ] }) ) ) .display(tnt.board.track.feature.block() .foreground_color("blue") .index(function (d) { return d.start; })); myBoard .add_track(location_track) .add_track(block_track); myBoard.start(); {% endhighlight %}
