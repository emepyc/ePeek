The Buttons-based theme

Overview

This example shows another way of panning and zooming the genome browser by using the left, right and zoom methods exposed by the epeek.genome API

Notice that, in this example, controlling panning and zooming directly via mouse gestures in the plug-in like in the minimal theme is also possible by pressing the button with a crossed-out hand icon. This feature can be switched on/off at any moment by calling the genome_browser.allow_drag method on the theme object.

A slider to control the sensibility of the buttons has also been added. When set to 0, no change occurs when buttons are pressed. When set to 1, panning/zooming occurs at 100% of the current coordinates.
