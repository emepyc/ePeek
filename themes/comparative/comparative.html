The Comparative Theme

Overview

Select different genes and species

In this theme two genome browsers showing the same gene in two different species are displayed. It uses buttons to control the panning and zooming as in the buttons theme. In this theme buttons to control both genome browsers at the same time and each of them independently are displayed.

Check the theme's source code

Because this theme displays more than one genome browsers, its "signature" is a bit different from the general pattern explained for other themes. In this case, instead of accepting a genome browser and a div element, it accepts an array of genome browsers as its first argument. So, the theme is initialized as follows:


var gB1 = epeek().species("human").gene(gene).allow_drag(false);
var gB2 = epeek().species("mouse").gene(gene).allow_drag(false);
gBTheme = epeek_theme();
gBTheme([gB1, gB2], document.getElementById("genomeBrowser"));

While the theme itself looks like:


var epeek_theme = function() {
    "use strict";
    // theme variables here
