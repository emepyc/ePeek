Themes

Overview

e!Peek themes surround and interact with the e!Peek core plug-in. The themes shown here are ready to use in your web page. They may also serve as examples of how to develop your own theme.

To use one of the following themes use the following code:

<script src="http://www.ebi.ac.uk/~mp/ePeek/v1/lib/ePeek.min.js" charset="utf-8"></script>
<script src="http://www.ebi.ac.uk/~mp/ePeek/v1/lib/themes/<theme_name>.min.js" charset="utf-8"></script>

var gB = epeek.genome().width(920); // other methods can be included here
var gBTheme = epeek_theme(); // other methods can be included here
gBTheme(gB, document.getElementById(<DOM_element_id>);

The themes may also have an external API to interact with (see for example the compact theme below)

Most of the themes shown below use the d3.js library for any DOM manipulation. This is not a requirement for any theme since the same results can be achieved with any other framework like, for example, jQuery

Basic Themes

These themes are examples that illustrate specific aspects of the genome core plug-in API

  • Minimal : Shows the minimum scheleton needed to build custom themes
  • Tooltips : Shows how to integrate tooltips when genes are selected
  • Legend : A simple example of how to add a legend to the genome browser
  • Pin : Shows how to display custom data in the genome browser
  • Buttons : Shows how to control a custom theme with buttons
  • Resizable-div : A more advanced examples of how to resize the genome browser

Advanced Themes

Themes themes show more advanced features of the genome core plug-in API

  • Comparative : Shows how to compare regions between two species
  • Compact : Shows a more functional theme with links, navigation and search capabilities
  • Orthology Tree : Shows a more sophisticated theme with orthologues displayed on a tree
  • Mobile : Shows a functional theme for iOS and Android
