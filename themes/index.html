Overview

TnT themes create final visualizations of trees and annotation tracks using the TnT library. TnT themes can be seen as reusable examples of tree visualizations. The themes shown here are ready to use in any web page. They may also serve as examples of how to develop your own theme

To use one of the following themes you have to include something like this in your web page:

{% highlight html linenos %} {% endhighlight %} {% highlight javascript linenos %} var t = tnt.tree() .data(epeek.tree.parse_newick(newick)) // newick is a tree string in newick format .layout(epeek.tree.layout.vertical()); // other methods can be included here var theme = tnt_theme(); // other methods can be included here theme (t, document.getElementById('DOM_element_id'); {% endhighlight %}

Themes may have also an external API to interact with. Some examples are given below

Most of the themes shown below use the D3.js library for DOM manipulation, ajax calls, etc. This is just for convenience, since D3.js is used by the TnT library for visualization. When building your own themes, though, you can use any library you feel comfortable with for that purposes (like jQuery)
