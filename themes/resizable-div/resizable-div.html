The Resizable Theme

Overview

This theme illustrates how the genome_browser.width and genome_browser.height methods can affect the dimensions of the genome browser dynamically. This is specially useful in the mobile theme that needs to adapt its size with changes in orientation of the mobile device. An example of usage of this property could be:


window.onorientationchange = function() {
   var w = document.documentElement.clientWidth;
   gBrowser.width(w-20);
};
