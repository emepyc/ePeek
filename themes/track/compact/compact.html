The Compact Theme

Overview

This example shows a more sophisticated theme that includes many others capabilities like searching, jumping to orthologues/paralogues, etc... It also includes links to the Ensembl browser and links to the mobile theme via a QR-code

The structure of the theme is exactly the same as we have seen in the other examples

API

This theme also defines its own basic API to control some of the themes parameters:

# theme.title ([string])

This method sets/gets the title to be displayed in the theme

# theme.show_title (<boolean>)

This method is used to set the visibility of the title. By default it is set to true

# theme.show_options (<boolean>)

This method is used to determine if the theme's options are displayed or not. By default the options are displayed

# theme.show_links (<boolean>)

This method is used to determine if the theme's links bottom panel is displayed. By default it is set to true
