The Legend Theme

Overview

Checkout the full code in github

This example shows an easy way of inserting a simple legend explaining the type of the displayed genes. This legend is dynamic in the sense that only shows the type of genes currently displayed. This gene type is based on the "biotype" of the genes and can be one of "protein_coding", "pseudogene", "processed transcript", "ncRNA" or "antisense"

Check the epeek.rest API

In this example, the legend needs to be updated every time a panning/zooming event is fired. To do this, the genome_browser.genes_callback method is used. This callback is called every time the genome browser plug-in queries the /feature/region/:species/:region endpoint of the Ensembl REST API
